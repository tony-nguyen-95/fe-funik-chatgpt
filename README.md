## Cài đặt

Clone project về máy

```bash
git clone https://github.com/tony-nguyen-95/fe-funik-chatgpt
```

Cài đặt các gói phụ trợ cần thiết

```bash
yarn
```

## Chạy project (development)

```bash
yarn start
```

## Build project (production)

```bash
yarn build
```
